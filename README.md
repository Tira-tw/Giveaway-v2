## 安裝

 **1.** 安裝 [node.js v16+](https://nodejs.org/) 

 **2.** 下載這個`檔案`並解壓 | 或 `git clone https://github.com/Tira-tw/Giveaway-v2/` 

 **3.**  **`npm install`** | **`npm install discord.js enmap`**

 **4** `config.json`添加Token or 使用 `.env`

 **5.** 啟動指令 :  **`node bot.js`**!
